# QMSHE 普通图/超图多随机种子与微调消融结论

## 实验设置

- 数据：HotpotQA 前 500 条，按 example ID 哈希固定划分为 362 train、75 validation、63 test。
- 随机种子：13、42、73；三个 seed 共享完全相同的 train/validation/test。
- 系统：Vanilla Dense、QMSHE 超图、Entity-Relation 普通图、Reified-Fact 普通图。
- 消融：base/tuned embedding × base/tuned reranker，共 4 格；总计 3,024 条系统级记录。
- embedding：BGE-M3 query-only LoRA，文档 encoder 始终冻结。
- reranker：BGE-reranker-v2-m3 LoRA，hard negatives 来自同题 distractor passages。

## 多 seed 主结果

数值为 test 集跨 seed 的 mean±sample SD。

| Embedding | Reranker | 系统 | Recall@20 | MRR |
|---|---|---|---:|---:|
| base | base | QMSHE 超图 | 0.6851±0.0252 | 0.8104±0.0142 |
| base | base | Entity-Relation | 0.8619±0.0092 | 0.8162±0.0049 |
| base | base | Reified-Fact | **0.9052±0.0265** | **0.8351±0.0014** |
| tuned | tuned | QMSHE 超图 | 0.7190±0.0109 | 0.8633±0.0170 |
| tuned | tuned | Entity-Relation | 0.8672±0.0048 | 0.8629±0.0235 |
| tuned | tuned | Reified-Fact | **0.9083±0.0345** | **0.8877±0.0020** |

## 微调带来的独立变化

相对 base embedding + base reranker：

| 系统 | Embedding-only ΔR@20 | Reranker-only ΔR@20 | Reranker-only ΔMRR | Joint ΔR@20 |
|---|---:|---:|---:|---:|
| QMSHE 超图 | **+0.0340** | +0.0000 | +0.0469 | +0.0340 |
| Entity-Relation | -0.0097 | +0.0123 | +0.0495 | +0.0053 |
| Reified-Fact | -0.0031 | +0.0088 | **+0.0526** | +0.0031 |

主要观察：

1. Query-LoRA 的 Recall 收益集中在超图（+3.40 个百分点），对两个普通图没有稳定提升。
2. reranker LoRA 的主要价值是把正确证据排到更前面：三个图系统 MRR 均提高约 4.7–5.3 个百分点。
3. Reified-Fact 的 Recall 已经很高，embedding 微调存在饱和现象；tuned reranker 单独使用时达到最高 Recall@20 0.9140。
4. 全微调时 Reified-Fact 同时取得最高 Recall@20 0.9083 和最高 MRR 0.8877。

## 普通图与超图的显著性

以问题为配对单位、先跨 seed 平均，再进行 10,000 次随机化检验和 paired bootstrap：

- base/base：Entity-Relation − Hypergraph 的 Recall@20 差为 +0.1768，95% CI [0.1102, 0.2443]，p=0.0001。
- base/base：Reified-Fact − Hypergraph 的差为 +0.2201，95% CI [0.1633, 0.2796]，p=0.0001。
- tuned/tuned：Entity-Relation − Hypergraph 的差为 +0.1481，95% CI [0.0825, 0.2152]，p=0.0001。
- tuned/tuned：Reified-Fact − Hypergraph 的差为 +0.1892，95% CI [0.1359, 0.2466]，p=0.0001。

在当前固定测试集上，普通图的 Recall 优势不是单一随机种子造成的；Reified-Fact 是综合最强链路。

## 训练阶段验证结果

- 三个 embedding seed 的 Recall@20 均从 0.7744 提升到 0.8067；最佳 epoch 都为 1。
- 三个 reranker seed 的 Recall@20 从 0.9000 分别提升到 0.9289、0.9356、0.9322；最佳 epoch 都为 3。
- reranker MRR 从 0.7819 分别提升到 0.8398、0.8389、0.8445。

## 适用边界

这些结果比原 50 条确定性编码器 smoke test 更可靠，但仍只覆盖 HotpotQA 前 500 条中的 63 条固定测试问题。
它证明了当前实现与该数据设定下的差异，不等价于跨数据集或领域材料语料的最终论文结论。下一步应在
2WikiMultiHopQA、MuSiQue 和真实领域语料上复现同一固定协议，并报告答案级 EM/F1 与生成忠实度。
