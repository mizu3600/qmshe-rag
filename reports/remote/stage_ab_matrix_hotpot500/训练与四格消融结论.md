# Stage A / Stage B 训练与四格消融结论

## 实验设置

- 数据：HotpotQA distractor 前 500 条，固定划分为 362 条训练、75 条验证、63 条独立测试。
- 随机种子：13、42、73。
- 结构：QMSHE 超图、Entity-Relation 普通图、Reified-Fact 普通图。
- Stage A：训练可迁移的多频带谱投影、滤波器和关系门控，用验证 Recall@20 优先、MRR 次优选择 checkpoint。
- Stage B：冻结 Stage A，对 BGE-M3 查询编码器进行 LoRA 微调，梯度穿过种子池化、Query Gate 与谱检索得分。
- Reranker：所有单元固定使用已微调的 BGE reranker，不把 reranker 变化混入 Stage A/B 差异。
- 评测规模：3 结构 × 4 单元 × 3 种子 × 63 测试样本 = 2268 条记录。

## 结论

1. **Stage A 已经是有效模块，不应再被排除在最终实验之外。** Stage A-only 使 Recall@20 相对 A0B0 分别提升：超图 +0.1342、Entity-Relation +0.0711、Reified-Fact +0.0265。
2. **完整 A1B1 对 Recall@20 的总增益**：超图 +0.1717、Entity-Relation +0.0737、Reified-Fact +0.0322。超图从 Stage A/B 训练中获益最大，明显缩小了与普通图的差距。
3. **最高 Recall@20 仍是 Entity-Relation A1B1：0.9480**；Reified-Fact A1B1 为 0.9462，两者非常接近。超图 A1B1 为 0.8568。
4. **Stage B 单独效果依赖结构。** 超图 A0B1 的 Recall@20 增加 +0.0340；Entity-Relation 和 Reified-Fact 分别为 -0.0071 和 -0.0057。但与已训练 Stage A 结合后，三者的 A1B1 均高于 A0B0。
5. **Recall 与首位排序不完全同步。** Reified-Fact 的最高 MRR 是 A0B0/A0B1 的 0.8878，而最高 Recall@20 是 A1B1 的 0.9462；选型时应按业务是更关心“找全证据”还是“第一个结果”决定。

## 解读边界

- 这是“顺序训练链路”的操作性四格消融，不是严格因果 2×2：A0B1 使用旧的 dense query LoRA，A1B1 使用在已冻结 Stage A 谱目标上训练的结构专属 LoRA。因此 `interaction` 可用于工程比较，不应解读为纯因果交互效应。
- 当前数据规模是 500 条 HotpotQA 中的 63 条固定测试集，结论应在更大规模与领域数据上再验证。
- `Accuracy/Hit@10` 表示 top-10 至少命中一条相关证据；`Complete@20` 是 top-20 覆盖全部 gold 证据的严格准确率。
